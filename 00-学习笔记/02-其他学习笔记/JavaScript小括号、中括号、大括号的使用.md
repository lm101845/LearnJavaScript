# 引言

代码是字符串，字符串就要被复制。显然在把代码从一个地方复制/剪切到另一个地方的时候，使用大括号+分号或者类似的结构的语言可以保持大概的语义不变。

无实际意义的字符，说白了就是增加可读性。
